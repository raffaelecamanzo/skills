# Sprint Log Template

Use this template for the output file `docs/planning/sprint-log.md`. This document tracks all sprint increments across the project — created on Sprint 1, updated on each subsequent sprint.

```markdown
# Sprint Log — [System/Feature Name]

> Source: docs/planning/journal.md
> Author: Sprint Planner (AI-assisted)
> Last Updated: [date]

## Overview

| Metric | Value |
|--------|-------|
| Total sprints | [N] |
| Current sprint | Sprint [N] |
| Stories planned | [N] of [total] |
| Stories completed | [N] |

## Sprint History

### Sprint 1: [Title]
- **Date:** [YYYY-MM-DD]
- **Status:** PLANNED | SELECTED | READY-FOR-REVIEW | DONE
- **Objective:** [1-sentence objective]
- **Stories:** S-NNN, S-NNN, S-NNN
- **References:**
  - SRS: docs/specs/software-spec.md [relevant sections]
  - Architecture: docs/specs/architecture.md [relevant sections]
  - Frontend Design: docs/specs/frontend-design.md (if applicable)
- **Sprint Document:** [sprints/sprint-1.md](sprints/sprint-1.md)

---

[...append new sprint entries...]
```
